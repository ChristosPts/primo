Lists

Group together related pieces of information

Some simple list styles for ordered and unordered lists.

Example #1

Use li-group-* classes on ul or ol tags to add the same style to every list item.

    Disc List
  • List item 1
  • List item 2
  • List item 3
    Square List
  • List item 1
  • List item 2
  • List item 3
    Check List
  • List item 1
  • List item 2
  • List item 3
    X List
  • List item 1
  • List item 2
  • List item 3
    Arrow List
  • List item 1
  • List item 2
  • List item 3
                
<ul class="li-group-disc"> 
    <div class="fw-bold">Disc List</div>         
        <li>List item 1</li>
        <li>List item 2</li>
        <li>List item 3</li>
    </ul>

<ul class="li-group-square"> 
    <div class="fw-bold">Square List</div>         
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

<ul class="li-group-check"> 
    <div class="fw-bold">Check List</div>         
        <li>List item 1</li>
        <li>List item 2</li>
        <li>List item 3</li>
    </ul>
    
<ul class="li-group-x"> 
    <div class="fw-bold">X List</div>         
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
                    
<ul class="li-group-arrow"> 
    <div class="fw-bold">Arrow List</div>         
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

Example #2

Use li-* classes to add bullet styles to each list item individually.

  1. Disc list item
  2. Square list item
  3. Check list item
  4. X list item
  5. Arrow list item
                
<ol class="mt-1"> 
    <li class="li-disc">Disc list item </li>
    <li class="li-square">Square list item </li>
    <li class="li-check">Check list item </li>
    <li class="li-x">X list item </li>
    <li class="li-arrow">Arrow list item </li>
</ol>
