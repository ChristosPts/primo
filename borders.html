Borders

Use border classes to style border colors and radius

  • The br-* classes are used to style border radius.
  • The border-* classes are used to style border colors.

Border positions

Use border-{color} to add a border or use -t , -r , -b or -l to add a border only to a specific side.

                
<div class="border-black"></div>
<div class="border-t-black"></div>
<div class="border-r-black"></div>
<div class="border-b-black"></div>      
<div class="border-l-black"></div>

Widths

Use lg or sm to change border widths.

                
<div class="border-black"></div>
<div class="border-lg-black"></div>
<div class="border-sm-black"></div>      
<div class="border-t-black"></div>
<div class="border-lg-r-black"></div>      
<div class="border-sm-l-black"></div>

Colors

Use any of the primary color classes with border-*.

                
<div class="border-primary"></div>
<div class="border-secondary"></div>
<div class="border-info"></div>
<div class="border-blue"></div>      
<div class="border-red"></div>
<div class="border-yellow"></div>
<div class="border-t-green"></div>
<div class="border-r-orange"></div>
<div class="border-b-purple"></div>      
<div class="border-l-black"></div>
<div class="border-white"></div>

Border radius

Use br-* with any border-* classes to add border radius

                
<div class="br-sm"></div>
<div class="br-mdk"></div>
<div class="br-lg"></div>      
<div class="br-full"></div>

Positioned radius

Use br-* with -t , -r , -b or -l to choose the border radius position.

                
<div class="br-t-sm"></div>
<div class="br-r-md"></div>
<div class="br-b-lg"></div>      
<div class="br-l-full"></div>
