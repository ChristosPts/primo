Font sizes

There are 5 font sizes with the default font size (font-size-mb) being 16px.

font-size-sm has a size of 12px
font-size-md has a size of 16px
If a font-size-* isn't assigned then text will have a default font size of 16px
font-size-lg has a size of 24px
font-size-xl has a size of 32px
font-size-xxl has a size of 48px

Headings

All 6 headings are available as tags or classes. The heading tags are also bolder than the classes.

h1 tags have a size of 46px

h1 classes have a size of 46px

h2 tags have a size of 40px

h2 classes have a size of 40px

h3 tags have a size of 34px

h3 classes have a size of 34px

h4 tags have a size of 28px

h4 classes have a size of 28px
h5 tags have a size of 22px
h5 classes have a size of 22px
