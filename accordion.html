Accordions

Create vertically collapsing accordions

  • Use the checked attribute on the input tag to keep one of the accordions open on load.
  • In each accordion the id attribute of the input tag corresponds to the for attribute of the label tag. These attributes need to match, in order to open the correct accordion.
