Spacing

Primo includes premade classes for paddings, margins and gaps.

Margins & Paddings

The classes range from 0 to 6 rem, with the base gap being 0.75rem

The class syntax is {property}{side}-{size}

    Properties
  • m - for margins
  • p - for paddings
    Sides
  • t - for top margin/padding
  • b - for bottom margin/padding
  • r - for right margin/padding
  • l - for left margin/padding
  • blank - for all sides
    Sizes
  • 0 - to remove margin/padding
  • 1 - for .75rem margin/padding
  • 2 - for 1.5rem margin/padding
  • 3 - for 3rem margin/padding
  • 4 - for 4.5rem margin/padding
  • 5 - for 6rem margin/padding
  • auto - to set the margin to auto

Examples

  • mt-4 - for a top margin of size 4
  • pl-3 - for padding left with a size of 3
  • p-5 - for padding with a size of 5
  • m-0 - remove all margins
