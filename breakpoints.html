Breakpoints

Breakpoints are pixel values defined in CSS. When a responsive website reaches those pixel values, a transformation occurs changing the layout of the website.They are often used in conjunction with containers and columns, creating an optimal user experience.

Available Breakpoints

Theres a total of six default breakpoints

Breakpoint Prefix Dimensions Devices
Extra Small -xs* < 576px Phones
Small -sm* ≥ 576px Tablets
Medium -md* ≥ 768px Laptops/Tablets
Large -lg* ≥ 992px Desktops/Laptops
Extra Large -xl* ≥ 1200px Desktops/Laptops
Extra Extra Large -xxl* ≥ 1400px Desktops
