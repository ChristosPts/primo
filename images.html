Images

Images are very important to beautify as well as to depict many complex concepts in simple way on your web page.

Responsive images

Apply max-width: 100%; and height: auto; with the image class so that images can scale with the parent element.

                    
 <img src="...." alt="...." class="image">

Thumnbails

Use the thumnbail class give a max height and width of 200px.

                    
 <img src="...." alt="...." class="thumbnail">

Customization

Use border and br classes to customize your images.

                    
<div class="col-*">
    <img src="...." alt="...." class="thumbnail border-black">
</div>
<div class="col-*">
    <img src="...." alt="...." class="thumbnail border-lg-l-blue br-r-md">
</div>
<div class="col-*">
    <img src="...." alt="...." class="thumbnail border-lg-secondary br-md">
</div>
<div class="col-*">
    <img src="...." alt="...." class="thumbnail br-full">
</div>
