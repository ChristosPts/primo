Buttons

There are multiple button style ready to use.

Use the btn class to create a basic button or use one of the predefiend styles.

Example #1

Create solid colored buttons with btn-*.

                    
<button class="mt-1 btn-primary">Primary</button>
<button class="mt-1 btn-secondary text-white">Secondary</button>
<button class="mt-1 btn-info text-white">Info</button>
<button class="mt-1 btn-blue text-white">Blue</button>
<button class="mt-1 btn-red  text-white">Red</button>
<button class="mt-1 btn-yellow">Yellow</button>
<button class="mt-1 btn-green text-white">Green</button>
<button class="mt-1 btn-orange text-white">Orange</button>
<button class="mt-1 btn-purple text-white">Purple</button>
<button class="mt-1 btn-black text-white">Black</button>
<button class="mt-1 btn-white">White</button>

Example #2

Create buttons with colored outlines with btn-outlined-*.

                    
<button class="mt-1 btn-outlined-primary">Primary</button>
<button class="mt-1 btn-outlined-secondary text-hover-white">Secondary</button>
<button class="mt-1 btn-outlined-info">Info</button>
<button class="mt-1 btn-outlined-blue">Blue</button>
<button class="mt-1 btn-outlined-red">Red</button>
<button class="mt-1 btn-outlined-yellow">Yellow</button>
<button class="mt-1 btn-outlined-green">Green</button>
<button class="mt-1 btn-outlined-orange">Orange</button>
<button class="mt-1 btn-outlined-purple">Purple</button>
<button class="mt-1 btn-outlined-black text-hover-white">Black</button>
<button class="mt-1 btn-outlined-white bg-hover-secondary">White</button>

Example #3

The btn classes can be used with a tags with role="button" and input tags with type="button".

Link Link 
                    
<input class="mt-1 btn-secondary text-white" type="button" value="Input">
<a class="mt-1 btn-outlined-secondary text-hover-white" href="#" role="button">Link</a>
<input class="mt-1 btn-info text-hover-white" type="button" value="Input">
<a class="mt-1 btn-outlined-info text-hover-white" href="#" role="button">Link</a>

Example #4

Create small buttons with btn-sm or large buttons with btn-lg. 

                    
<button class="mt-1 btn-sm btn-secondary text-white ">Small Btn</button>
<button class="mt-1 btn-sm btn-outlined-info text-hover-white">Small Btn</button>
<button class="mt-1 btn-lg btn-secondary text-white ">Large Btn</button>
<button class="mt-1 btn-lg btn-outlined-info text-hover-white">Large Btn</button>
